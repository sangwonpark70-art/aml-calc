서류 확인 가이드

고객이 가져온 서류를 접수 전 검토할 때, 각 서류에서 반드시 확인할 지점을 모형으로 안내합니다. 붉은 표시(확인 포인트)를 현장에서 그대로 대조하세요.

📄 법인등기부등본 확인법

고객이 가져온 등기부등본에서 아래 3곳을 확인하세요.

등기사항전부증명서
확인 1제목이 "전부증명서"인지 — "일부증명서"는 접수 불가 (말소사항 포함 필수)
확인 2법인 구분 — "주식회사 / 재단법인 / 사단법인" 표기 확인 (유형 판별 근거)
확인 3하단 발행일자 — 오늘 기준 3개월 이내인지 계산
  • 3개월 경과 시: 명판(회사명) + 현재날짜 기재 + 직인 날인하면 인정 가능 (내부 기준)

📄 고유번호증 (비영리단체)

고유번호증
확인 1고유번호 중간 2자리가 "82·83·89" 등 — 비영리단체 번호대인지
확인 2단체명·대표자 — 실제소유자 확인서의 대표자와 일치하는지
  • 미등기 임의단체: 정관·회칙 + 대표자 선임 확인서류(의사록 등) 추가 필요

📄 주주명부 확인법

주 주 명 부
확인 1기준일(작성일) — 3개월 이내인지
확인 2지분율 합계 100% — 일부 주주 누락 여부
확인 3하단 회사명(명판)·대표이사 직인 — 날인 없는 명부는 접수 불가
  • 수기 작성 명부도 위 3요소가 있으면 유효 — 없으면 ② 표준양식 안내

법인설립허가증

  • 정관: 대표자(이사장) 선임 조항과 실제 대표자 일치 확인
